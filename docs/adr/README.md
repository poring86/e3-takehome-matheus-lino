# ADR Index

This folder contains Architecture Decision Records (ADRs) for long-lived technical decisions.

## How to use

- Write ADRs for decisions that shape architecture, boundaries, data flow, or platform standards.
- Keep operational timelines and validations in `NOTES.md`.
- Link related `NOTES.md` entries to the ADR number.
